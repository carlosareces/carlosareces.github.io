Racer

  • Descripción:

    RACER is a tableaux based theorem prover for descripition logics. It supports the description logic ALCQHIR+(D)- (called ALC-choir-D-minus), i.e. ALC plus qualified number restrictions, role hierarchies, inverse roles and transitive roles and a restricted form of concrete domains. The logic ALCQHIR+ (without concrete domains) is also called SHIQ. RACER is the first DL system for TBox and *ABox* reasoning in this logic. RACER is based on the unique name assumption for ABoxes. All standard inference services for TBoxes and ABoxes are supported.

    Haarslev and Möller, Description of the RACER System and its Applications

  • Desarrollado por: Volker Haarslev y Ralf Möller

Bliksem

  • Descripción:

    • Bliksem is a theorem prover that uses resolution with paramodulation. It is written in portable C. The purpose of Bliksem was to develope a theorem prover that is theoretically up to date, but which is also efficient on the technical level. It is able to find proofs in first order logic and in modal logics, by translation formulae into clauses. It supports different transformations to clausal normal form. Bliksem is intended both for stnad alone use, and for use in combination with an interactive theorem prover. In order to obtain this goal Bliksem is able to output full detailed proofs, which can be verified by another program, or translated into another calculus.

    de Nivelle, Bliksem 1.10, System Description

  • Desarrollado por: Hans de Nivelle
  • Download: http://www.mpi-sb.mpg.de/~bliksem
